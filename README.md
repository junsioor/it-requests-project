IT REQUESTS PROJECT

Це простий веб-додаток для створення та перегляду заявок в ІТ.

У цьому проєкті реалізовано можливість додавати заявки (наприклад, якщо не працює клавіатура або мишка) та переглядати їх список у браузері.
Проєкт складається з двох частин: серверної (Node.js) та клієнтської (React). Для того щоб подивитися як працює додаток, потрібно запустити обидві частини.
Спочатку потрібно запустити сервер. Для цього відкрити термінал, перейти в папку server, виконати команду npm install для встановлення залежностей, після чого запустити сервер командою node index.js. Якщо все зроблено правильно, в консолі з’явиться повідомлення, що сервер працює на http://localhost:5000
.
Далі потрібно відкрити другий термінал і перейти в папку client. Там також потрібно виконати npm install, а потім запустити клієнт командою npm start. Після цього автоматично відкриється браузер за адресою http://localhost:3000
.
Після запуску відкриється сторінка з формою. У поля потрібно ввести назву проблеми та опис, наприклад "Не працює мишка" і короткий опис. Після натискання кнопки заявка відправляється на сервер і одразу з’являється у списку.
Таким чином можна перевірити, що додаток працює: додати кілька заявок і переконатися, що вони відображаються на сторінці без перезавантаження.\
